You are working in a Social CRM system.

---

## 🎯 Goal

Ensure a new user performs their first meaningful action within 10–20 seconds.

---

## 🧠 CONTEXT

Users arrive with:

- No leads
- No setup
- No understanding

---

## 🧠 TASK

Design or improve a flow that pushes the user to:

👉 Add a lead
or
👉 Import leads

---

## 🔍 FOCUS

- Empty states
- CTA placement
- First screen experience
- Reducing friction

---

## ✅ ALLOWED

- Add banners
- Add CTA buttons
- Add guidance text
- Improve flow between pages

---

## 🚫 FORBIDDEN

- Backend changes
- Complex state logic
- New APIs

---

## 🧪 SUCCESS CRITERIA

User should:

- Understand what to do immediately
- Perform an action quickly
- See a result

---

## 🧾 OUTPUT

- Flow description
- UI changes
- Exact text (Hebrew)
- Where each element appears

---

Mindset:

"If the user doesn't act — the product fails"
